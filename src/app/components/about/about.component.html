Objetivo:

Projeto feito para exibir receitas de uma API aleatoriamente na página principal, tendo a opção de busca ao clicar em 'Buscar receita', e também exibir receitas similares.

Tenho um número "X" de requisições por dia; caso não exiba nenhuma receita, é por este motivo. Espere e tente outro dia, obrigado.
